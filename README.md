# لائحة الدراسات العليا - كلية الإمام مالك للشريعة والقانون

موقع تفاعلي للائحة الدراسات العليا بكلية الإمام مالك للشريعة والقانون - دبي.

## الرابط المباشر

🔗 **[https://elrashid.github.io/pgv4/](https://elrashid.github.io/pgv4/)**

## المميزات

- عرض اللائحة بتنسيق منظم مع فهرس جانبي للتنقل السريع
- دعم كامل للغة العربية واتجاه RTL
- عرض مرجعي جانبي لصفحات PDF الأصلية (زر أخضر أسفل يسار الشاشة)
- بحث فوري في جميع الأبواب والمواد
- تصميم متجاوب للجوال والطباعة
- تنقل بين الصفحات (السابق/التالي)
